## ðŸ§  About MASM 8086 Assembly Language

**Assembly language** is a low-level programming language that is **closely related to a computer's machine code**.  
It allows programmers to write instructions that the CPU executes directly, providing **control over hardware, memory, and registers**.

**MASM (Microsoft Macro Assembler)** is a widely used assembler for the **8086 microprocessor** family.  
It converts assembly code into executable machine code and supports macros, structured programming, and advanced instructions.

Key concepts covered in MASM 8086:

- **Data Movement Instructions:** MOV, XCHG, LEA  
- **Arithmetic Operations:** ADD, SUB, MUL, DIV, INC, DEC  
- **Logic Instructions:** AND, OR, XOR, NOT, TEST  
- **Control Flow:** JMP, JZ, JNZ, LOOP  
- **String Operations:** MOVS, LODS, STOS, CMPS, SCAS  
- **Interrupts & System Calls:** INT 21h, DOS functions  
- **Stack Operations:** PUSH, POP, CALL, RET
